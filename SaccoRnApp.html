Project Description

This project is built using React Native.
it has the following features:
  • Allows staff to register members
  • Allows staff to create group and joint accounts
  • Allows staff to set loan policices e.g fine, interest, loan duration , etc
  • Allows staff to approve loan requests based on previous loan payment history
  • Allows staff to proccess withraw request from group and personal accounts
  • Allows members to deposit to person, jambo account accounts using mpesa
  • Allows members to request for withdrawals,loans
  • Allows members to pay loans in their loans through mpesa stk push
  • Allows members to view balances and previous transactions from their respective accounts
